# 🧠 Python OOP NLP Toolkit

A desktop Natural Language Processing (NLP) application developed using **Python**, **Object-Oriented Programming (OOP)**, and **Tkinter**. The application provides an interactive graphical user interface (GUI) for performing NLP tasks through external NLP APIs.

This project was created while learning Python OOP, GUI development, API integration, and modular programming.

---

## 🚀 Features

- 🔐 User Login & Registration System
- 🖥️ Desktop GUI built with Tkinter
- 😊 Sentiment Analysis
- 🏷️ Named Entity Recognition (NER)
- 📂 Dictionary/JSON-based user data storage
- 🔗 NLP API Integration
- 📦 Modular Object-Oriented Project Structure

---

## 🛠️ Technologies Used

- Python 3
- Tkinter
- Object-Oriented Programming (OOP)
- NLPCloud API

- REST API

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Python-OOP-NLP-project.git
```

2. Install the required library

```bash
pip install nlpcloud
```

3. Add your NLP API key inside `myapi.py`

```python
API_KEY = "YOUR_API_KEY"
```

4. Run the application

```bash
python main.py
```
